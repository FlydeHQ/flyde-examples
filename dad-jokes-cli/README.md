# Flyde Example Project - Dad Jokes in your CLI

![Preview](/preview.gif)

CLI tool that tells dad jokes, with a slight delay between the joke's setup and punch-line.

- [DadJokes.io](https://dadjokes.io/) for the jokes

## Running it
1. `yarn`
2. `yarn joke`
